# Лабораторная работа 4

## Overview 
Эта программа анализирует данные про международные компании. Возможность данной программы состоит в том, 
что она умеет считывать данные из файла, фильтровать компании на основе числа работников, а также может сохранять рещультаты 
в другом файле.

## Usage
```bash
./bin/company_analyzer
# или
make run
```

## Additional Notes
Входной файл должен иметь следующий формат.
```text
Название,Сайт,Индекс,Страна,Область,Район,Город,Улица,Дом,Квартира,Лет на рынке,Сотрудников,Филиалов в Европе
Microsoft,www.microsoft.com,98052,USA,Washington,King,Redmond,Microsoft Way,1,,45,144000,15
Google,www.google.com,94043,USA,California,Santa Clara,Mountain View,Amphitheatre Parkway,1600,,22,135000,20
```
